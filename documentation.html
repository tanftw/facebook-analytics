1. Installation - top

Site Requirements

Facebook Analytics has tested and compatibility with

  • WordPress 4.0+
  • PHP 5.2+

Preinstall

  • Make sure you have backed up your database before install
  • It's recommended that you remove another Facebook Javascript SDK if exists
  • Facebook Analytics is an addon of Get Facebook Likes so make sure you have installed Get Facebook Likes before install this plugin. It's easy and available at WordPress plugin repository so you can install that plugin through the WordPress plugins screen directly.

Installation

  1. Install Get Facebook Likes plugin, for more information, please follow this guide
  2. Download facebook-analytics.zip and unzip the package
  3. Upload facebook-analytics folder to the wp-content/plugins/ directory
  4. Activate the plugin through the Plugins menu in WordPress
  5. Start using the plugin through the menu Settings\Get Facebook Likes

2. Configuration - top

Because we'll need to send data to Google Analytics right after users click Like or Comment. These following steps help you do that:

  1. In General Settings \ Mode, select Advanced
  2. Check Auto add JS SDK to wp_head checkbox (*)
  3. Optional Enter App ID and App Locale if you want

(*) In case you're already have another JS SDK installed, you can either uncheck Auto add JS SDK to wp_head option and follow the setup guide appear right after you uncheck or remove another JS SDK.

Facebook Analytics Settings

In this section, choose the action that you want to track and enter your Google Analytics ID. Leaves blank if you're already have tracking code installed

4. Uninstallation - top

Go to Plugins\Installed Plugins, locate to Facebook Analytics and click "Deactivate".
